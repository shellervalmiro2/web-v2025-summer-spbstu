##Задача 5.
---
Обеспечить переключение темного и светлого режимов 
Создайте список книг в магазине. Сверху справа расположите кнопку
переключения темы. Реализуйте логику для изменения режима отображения в
приложении с помощью Context API. 
Функциональность:
- кнопка меняет свое название (иконки) в зависимости от выбранного
режима; 
- для каждого цвета светлой темы, должен быть подобран альтернативный
цвет в темной теме. 
Примечание: Нельзя использовать дизайн библиотеки. Используемые
технологии согласовать с преподавателем (если такие будут)
